# Aplicación Notas

Aplicación Java con Spring Boot para la creación y consulta de notas.
Usando la arquitectura hexagonal (aplicación, dominio e infraestructura).

### Especificaciones técnicas
* Java 8
* Gradle

### Tecnologías usadas
* Spring Boot
* Lombok
* Spring JPA
